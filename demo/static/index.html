How It Works

Hilbert Curves

LOCI maps 4D coordinates (x, y, z, t) to a single integer via space-filling curves. This turns expensive multi-axis range queries into simple integer-set comparisons. A spatial bounding box becomes a single filter: hilbert_r4 IN {ids}.

Temporal Sharding

Vectors are automatically routed to time-partitioned collections (loci_0, loci_1, ...). Queries skip irrelevant epochs entirely. The async client searches all matching shards concurrently via asyncio.gather.

Novelty Scoring

Compare predicted future embeddings against stored memories. High distance = high novelty = something unexpected. Score 0.0 means "I've seen this exact situation before." Score 1.0 means "This is completely new territory."
