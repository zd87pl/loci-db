How It Works

Hilbert Curves

LOCI maps 4D coordinates to a single integer via space-filling curves. A bounding box becomes one filter: hilbert_r4 IN {ids}.

Temporal Sharding

Vectors are routed to time-partitioned collections. Queries skip irrelevant epochs. Async client searches matching shards concurrently.

Novelty Scoring

Compare predicted future embeddings against stored memories. High distance = high novelty = the robot sees something unexpected.
